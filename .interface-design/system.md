# Scapder Interface Design Memory

This project is a marketing landing page, so `interface-design` is not the primary design authority. Use root `system.md` as the source of truth.

## Direction
- Personality: monochrome, technical, precise, editorial.
- Foundation: Scapder-first AI/data-intelligence company landing.
- Depth: tonal layering and borders only.
- Typography: Geist Mono everywhere.
- Radius: 4px controls, 8px large containers max.

## Tokens
Use the semantic tokens defined in `../system.md`:
- canvas, paper, surface, surface-raised, surface-dim;
- ink, ink-soft, ink-muted;
- border, border-strong;
- inverse, inverse-ink;
- error.

## Consistency Contract
Before adding or changing UI, read `../system.md`. Do not introduce generic AI UI patterns, non-monochrome accents, fake data, fake logos, or undocumented component values.
